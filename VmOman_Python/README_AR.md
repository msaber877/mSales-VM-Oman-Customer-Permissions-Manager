# 📱 برنامج دمج بيانات mSales & تعديل أذونات التجار

## 🎯 نظرة عامة

برنامج احترافي لإدارة ودمج بيانات العملاء مع قائمة التجار ومعالجة أكواد الأذونات بشكل آلي وفعال، مع دعم كامل للتحكم بالأكواد ديناميكياً.

**الإصدار:** 3.1.0 (Dynamic Permission Codes Edition)
**الشركة:** EME International
**الفريق:** IT Support Team
**الترخيص:** VM-Oman Edition

---

## ✨ الميزات الجديدة والتحديثات في إصدار 3.1.0

1. **💾 إدارة ديناميكية لأكواد الأذونات:**
   - يتم تحميل الأكواد ووصفها ديناميكياً من ملف خارجي باسم `permission_codes.json`.
   - لا حاجة لتعديل الكود البرمجي أو إعادة بناء ملف الـ EXE عند إضافة أكواد جديدة أو تعديل الأكواد الحالية.
   
2. **➕ واجهة رسومية لإدارة الأكواد:**
   - إمكانية إضافة كود أذونات جديد مع الوصف الخاص به من داخل التطبيق مباشرة وحفظه تلقائياً.
   - إمكانية حذف الأكواد القديمة أو غير المستخدمة من القائمة.

3. **🔄 دمج ذكي متطور:**
   - دمج تلقائي بين ملفات CSV و Excel.
   - مطابقة دقيقة لأكواد التجار.
   - تحويل الفواصل تلقائياً (- إلى ,) لتوحيد تنسيق الأكواد.

4. **💾 تصدير بدون عمود الحالة:**
   - يتم تصدير النتائج النهائية خالية من عمود الحالة (STATUS) لتكون جاهزة للمطابقة والرفع مباشرة.

## 🚀 التشغيل والبدء السريع

### الخطوة 1️⃣: تشغيل التطبيق بالبيئة الافتراضية للنظام
افتح نافذة PowerShell أو Command Prompt في مجلد المشروع وشغّل الأمر التالي:
```powershell
& "C:\Users\mohamed_saber\AppData\Local\Programs\Python\Python312\python.exe" mSales_Data_Merger_FIXED.py
```

### الخطوة 2️⃣: استخدام البرنامج للدمج
1. **الشاشة الرئيسية:** اضغط على "▶ Start Processing".
2. **شاشة رفع الملفات:**
   - اختر ملف العملاء (CSV File) - مثل `Customers_TEMPLATE-33.csv`.
   - اختر ملف التجار (Excel File) - مثل `List-of-500-Dealers.xlsx`.
   - اضغط "▶ Next".
3. **شاشة النتائج:**
   - ستظهر لك إحصائيات المطابقة وجدول المعاينة باللون الأخضر للمتطابق والأحمر لغير الموجود.
   - لحفظ الملف مباشرة بدون تعديل، اضغط "💾 Save Results".
   - للتعديل الجماعي على الأكواد، اضغط "✏️ Edit Codes".

### الخطوة 3️⃣: إضافة أو تعديل أكواد الأذونات (Available Codes)
1. من شاشة التعديل الجماعي "Edit Codes"، ستظهر لك قائمة الأكواد في اليسار.
2. **لإضافة كود جديد:** اضغط على زر "➕ Add Code" واكتب الكود الجديد ووصفه ثم اضغط Save.
3. **لحذف كود:** حدد الكود المطلوب من القائمة ثم اضغط "🗑️ Delete Code".
4. يتم حفظ أي تعديل تلقائياً في ملف `permission_codes.json` لتظل متواجدة عند فتح التطبيق في المرات القادمة.

## 🔧 بناء ملف EXE للتطبيق (PyInstaller)

إذا أردت تشغيل البرنامج كملف تنفيذي مستقل (`.exe`) دون الحاجة لتواجد بايثون أو تشغيل سطر الأوامر:

1. ثبّت مكتبة pyinstaller:
   ```bash
   & "C:\Users\mohamed_saber\AppData\Local\Programs\Python\Python312\python.exe" -m pip install pyinstaller
   ```
2. ابنِ الملف التنفيذي باستخدام ملف الإعدادات الجاهز:
   ```bash
   & "C:\Users\mohamed_saber\AppData\Local\Programs\Python\Python312\python.exe" -m PyInstaller mSales_Data_Merger.spec
   ```
3. ستجد الملف التنفيذي الناتج في مجلد `dist/mSales_Data_Merger.exe`.
4. انسخ هذا الملف وضعه في أي مكان، وتأكد من وجود ملف `permission_codes.json` بجواره إذا أردت تخصيص الأكواد ديناميكياً.
